3.0h

8.6h

크고 작은 모든 장비 트러블(Incident)이 발생하는 총합의 평균 간격입니다.

4 현장 실무 적용 및 비즈니스 룰

상태 전환 시 혼동을 방지하기 위한 시스템 자동화 규칙 가이드입니다.

복합 상태 우선순위 룰

SEMI E10은 한 순간에 반드시 1개의 상태만 가져야 하므로, 상태가 겹칠 경우 엄격한 우선순위(Priority Rule)가 적용됩니다.

상황 A: 대기 중 고장 발생 (SBY + UDT)

장비가 자재를 기다리며 쉬고(SBY) 있었는데, 정전으로 장비가 꺼지거나 내부 알람이 발생했습니다.

➔ UDT 우선 기록 (고장 상태가 대기 상태보다 우선함)
상황 B: 정비 중 실수 파손 (SDT + UDT)

엔지니어가 예방 정비(SDT) 중 실수로 부품을 파손하여 예상 수리 시간을 초과했습니다.

➔ 원래 계획된 시간까지만 SDT, 초과분부터는 UDT 전환

Failure vs Assist 명확한 구분

장비 중단 사유를 명확히 하여 신뢰성 지표(MTBF)의 왜곡을 막기 위해 팹(Fab) 내부의 엄격한 구분이 필요합니다.

구분 기준 Failure (UDT 처리) Assist (Non-UDT)
복구 주체 전문 정비 엔지니어 (Equipment Eng.) 오퍼레이터 또는 시스템 자동 리셋
물리적 조치 부품의 교체, S/W 재설치, 장비 분해 단순 화면 터치, 걸린 웨이퍼 수작업 제거
소요 시간 보통 6분(또는 10분) 초과 소요 지정된 짧은 시간 내에 복구 완료
사후 검증 가동 전 파티클/더미 테스트 필수 수반 별도 테스트 없이 즉시 원래 공정 재개
