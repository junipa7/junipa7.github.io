🔒 [특별편] 완벽한 SSL 인증서(HTTPS) 발급 및 자동 갱신 가이드

💡 이 단계의 핵심 목표

안드로이드 스마트폰(Termux) 환경에서 외부 도메인(DuckDNS)에 군사급 암호화(HTTPS)를 적용하기 위해 acme.sh 스크립트를 설치하고 인증서를 발급받습니다. 한 번 설정해 두면 평생 알아서 인증서를 갱신하는 완전 자동화 시스템을 구축하는 것이 목표입니다.

1. 인증서 발급 및 갱신을 위한 필수 패키지 설치

Termux 환경 특성상 인증서 발급 스크립트가 실행되기 위해 꼭 필요한 도구들(스케줄러, 암호화 도구)을 먼저 설치하여 에러를 원천 차단합니다.

# 1. 인증서 자동 갱신을 담당할 스케줄러(cronie) 설치
pkg install cronie -y

# 2. 스케줄러 데몬 백그라운드 가동
crond

# 3. 인증서 키 생성을 위한 OpenSSL 실제 실행 도구 설치
pkg install openssl-tool -y

2. acme.sh 스크립트 설치

인증서를 무료로 발급받고 관리해 주는 똑똑한 자동화 스크립트를 설치합니다.

# acme.sh 다운로드 및 설치
curl https://get.acme.sh | sh

# 설치된 스크립트를 현재 터미널 환경에 즉시 적용
source ~/.bashrc

3. ZeroSSL 이메일 등록 (최초 1회 필수)

최근 acme.sh의 기본 인증 기관이 Let's Encrypt에서 ZeroSSL로 변경되었습니다. ZeroSSL은 인증서 만료 안내 등을 위해 이메일 계정 연동을 필수로 요구합니다.

⚠️ 이메일 주소 변경 필수

아래 명령어에서 본인실제이메일@gmail.com 부분을 본인이 주로 사용하는 이메일로 반드시 수정하여 입력하세요.

# ZeroSSL 계정 생성 및 이메일 등록
~/.acme.sh/acme.sh --register-account -m 본인실제이메일@gmail.com

4. DuckDNS 토큰 입력 및 SSL 인증서 발급

DuckDNS 홈페이지에서 확인한 본인의 고유 Token 값을 서버에 입력하여 소유권을 증명하고, 가볍고 빠른 ECC 방식의 인증서를 발급받습니다.

⚠️ 정보 수정 필수

토큰 값(여기에_진짜_토큰값을_입력하세요)과 도메인 주소(topooh.duckdns.org)를 본인의 실제 정보로 변경한 뒤 실행하세요.

# DuckDNS 토큰 등록 (따옴표 유지)
export DuckDNS_Token="여기에_진짜_토큰값을_입력하세요"

# 인증서 발급 실행 (약 10초 ~ 1분 소요)
~/.acme.sh/acme.sh --issue --dns dns_duckdns -d topooh.duckdns.org --ecc
✅ 발급 성공 확인

화면에 무수히 많은 통신 기록이 지나간 후, 초록색 글씨로 Cert success! 문구가 나타나면 성공입니다. 인증서 파일은 ~/.acme.sh/topooh.duckdns.org_ecc/ 경로에 안전하게 저장됩니다.

5. 영구적인 인증서 자동 갱신 시스템 구축 (끝판왕)

ZeroSSL 인증서의 유효기간은 90일입니다. 앞서 1단계에서 설치한 스케줄러(cron)가 매일 밤 날짜를 체크하여, 만료 30일 전에 자동으로 새 인증서로 교체하도록 설정되어 있습니다.

🤖 내 눈으로 자동 갱신 알람 확인하기

터미널에 아래 명령어를 입력해 보세요.

crontab -l

화면에 acme.sh --cron ... 으로 시작하는 문구가 출력된다면, 자동 갱신 알람이 완벽하게 맞춰진 것입니다!

서버 재부팅 시 스케줄러 자동 실행 보장하기

스마트폰이 재부팅되면 스케줄러(알람 시계)도 꺼지기 때문에 자동 갱신이 멈출 수 있습니다. 이를 방지하기 위해, 안드로이드가 켜질 때 스케줄러를 알아서 다시 켜도록 부팅 스크립트에 명령어를 한 줄 추가합니다.

# Nextcloud 자동 부팅 스크립트 맨 아래에 crond 실행 명령을 강제로 덧붙임
echo "crond" >> ~/.termux/boot/start-nextcloud.sh

🎉 인증서 유지보수 영구 면제!

수고하셨습니다. 이제 서버는 평생 알아서 인증서를 갱신합니다. 사용자는 인증서 만료 걱정 없이 안심하고 자신만의 클라우드 환경을 즐기기만 하면 됩니다!
